But I don’t want to spend money, does UNLV provide anything? Is it free? Is it possibly online?

Well lucky you, UNLV does. Kinda.

  1. First go to www.google.com
  2. Type in UNLV Lied Library
  3. Click on the lied library link: https://www.library.unlv.edu/
  4. Open a tab with the databases link. You should have 2 tabs now with the lied library page
  5. In the first page, use the browser to type in pcb design
  6. In the second page which has our entire database access, find SpringerLink and click on that
  7. In the SpringerLink page, search pcb design

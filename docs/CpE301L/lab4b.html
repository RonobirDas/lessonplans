Important Note on Dynamic Memory management

Try not to leave memory management of pointers and arrays or objects in general to the GCC compiler. While it will try its best, it is better to work with hard coded limits, and monitor memory usage. Make a memory management function that monitors usage, statically creates memory blocks at runtime and cleans after an object is no longer in use freeing up that space.
